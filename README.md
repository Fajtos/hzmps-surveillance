# HZMPS Surveillance: Bayesian State-Space Hurdle Model for Disease Surveillance

This repository contains code, data, and results for a Statistical framework for disease surveillance that correctly separates true outbreak signals from persistent reporting differences between regions/countries.


## Contents

- `simulation  : Simulation study code and results (7 scenarios, 100 reps)
- `empirical` : COVID-19 and dengue data analysis code
- `figures  : All 11 publication-ready figures


## Requirements

- Python 3.10+
- CmdStanPy
- Pandas, NumPy, Matplotlib, Seaborn
